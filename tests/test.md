# 测试报告
主要针对NDSL网络库和muduo网络库中的线程池进行压力测试

## 测试环境
系统: Ubuntu 16.04LTS

CPU: Intel(R) Core(TM) i7-4710MQ CPU @ 2.50GHz

CPU核心数: 8

内存: 8G
